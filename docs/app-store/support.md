# Twilight Compass / 光影罗盘 Support

## English

### App version

1.0

### What the app does

- Calculates blue hour and golden hour on-device from your current coordinates
- Shows sun, moon, and compass bearings
- Offers optional local alerts
- Supports Chinese and English UI

### Troubleshooting

1. Time or bearings are not updating
Make sure location access is allowed, then tap `Refresh location` in Settings.

2. The compass map base is missing
The map base depends on Apple MapKit tiles. The app can continue calculating on-device while offline, but the map base may not appear.

3. Alerts are not firing
Confirm that notification permission is enabled in iOS Settings and that twilight alerts are turned on in the app.

4. The language did not change
Change the language in Settings. If the interface still looks stale, fully quit and reopen the app.

## 简体中文

### 适用版本

1.0

### App 功能

- 根据当前经纬度在本地计算蓝调与金调
- 显示太阳、月亮与罗盘方位
- 提供可选本地提醒
- 支持中文与英文界面

### 常见问题

1. 时间或方位不更新
请确认已授予定位权限，并在设置页点击 `刷新定位`。

2. 罗盘底图没有显示
地图底图依赖 Apple MapKit 网络瓦片。离线或网络受限时，App 仍可继续本地计算，只是不显示地图底图。

3. 提醒没有触发
请确认 iOS 通知权限已开启，并检查 App 内是否已打开时段提醒。

4. 语言没有切换
请在设置页修改语言选项；如果界面仍未刷新，完全退出并重新打开 App。
